📡 API Documentation

Image Classification API using Keras/TensorFlow Model

🚀 Quick Start

Send a POST request with an image file to get classification predictions.

curl -X POST -F "image=@your_image.jpg" http://localhost:8000/api/predict/

📍 Endpoints

POST /api/predict/

Classify an image using the trained model.

Request Parameters

Parameter Type Required Description
image File Required Image file (jpg, jpeg, png, bmp, gif)
top_k Integer Optional Number of top predictions to return (default: 5)

Example Requests

cURL:

curl -X POST \
  -F "image=@cat.jpg" \
  http://localhost:8000/api/predict/

cURL with top_k:

curl -X POST \
  -F "image=@cat.jpg" \
  -F "top_k=3" \
  http://localhost:8000/api/predict/

Python (requests):

import requests

url = "http://localhost:8000/api/predict/"
files = {'image': open('cat.jpg', 'rb')}

response = requests.post(url, files=files)
print(response.json())

Python with top_k:

import requests

url = "http://localhost:8000/api/predict/"
files = {'image': open('cat.jpg', 'rb')}
data = {'top_k': 3}

response = requests.post(url, files=files, data=data)
print(response.json())

JavaScript (Fetch API):

const formData = new FormData();
formData.append('image', fileInput.files[0]);

fetch('http://localhost:8000/api/predict/', {
    method: 'POST',
    body: formData
})
.then(response => response.json())
.then(data => console.log(data));

Response Fields

Field Type Description
success Boolean Whether the request was successful
predictions Array List of predictions with class name and confidence
predictions[].class String Predicted class name
predictions[].confidence Float Confidence score (0-1)
predictions[].confidence_percent String Confidence as percentage
top_prediction Object The highest confidence prediction

{
  "error": "No image provided"
}

{
  "error": "Invalid file type. Allowed: .jpg, .jpeg, .png, .bmp, .gif"

{
  "error": "Error message details"

📝 Notes

CSRF Protection: The /api/predict/ endpoint is CSRF-exempt for easy API access. For production, implement proper authentication.

File Size: Maximum file size depends on your Django settings. Default is typically 2.5 MB.

Image Formats: Supported formats: JPG, JPEG, PNG, BMP, GIF. Images are automatically resized to match model input dimensions.

🔧 Technical Details

Model Information

  • Format: Keras (.keras)
  • Framework: TensorFlow
  • Input Processing: Automatic resize and normalization
  • Output: Softmax probabilities

Rate Limits

Currently no rate limiting is implemented. For production use, consider adding rate limiting middleware.

CORS

CORS is not configured by default. To enable cross-origin requests, add django-cors-headers to your project.

💡 Examples by Language

PowerShell

$uri = "http://localhost:8000/api/predict/"
$filePath = "C:\path\to\image.jpg"
$form = @{
    image = Get-Item -Path $filePath
}
Invoke-RestMethod -Uri $uri -Method Post -Form $form

Node.js

const FormData = require('form-data');
const fs = require('fs');
const axios = require('axios');

const form = new FormData();
form.append('image', fs.createReadStream('image.jpg'));

axios.post('http://localhost:8000/api/predict/', form, {
    headers: form.getHeaders()
})
.then(response => console.log(response.data))
.catch(error => console.error(error));

PHP

<?php
$url = 'http://localhost:8000/api/predict/';
$file = new CURLFile('image.jpg');

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, ['image' => $file]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
curl_close($ch);

echo $response;
?>
